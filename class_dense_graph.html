Construct a new Dense Graph object from VIDs iterators.

Construct a new Dense Graph object from a pair of VIDs iterators. Let n be the size of the sequence of VIDs V, each VID in V is assumed to be (1) unique and (2) between 0 and n-1.

The sequence is not required to be ordered, nor locally stored.

Construct a new Dense Graph object from EIDs iterators.

Construct a new Dense Graph object from a pair of EIDs iterators. Each EID in the sequence of EIDs is assumed to be unique. Let V be the hypothetical sequence of VIDs given the sequence of EIDs. Let n be the size of the sequence of VIDs V, each VID in V is assumed to be (1) unique and (2) between 0 and n-1.

The sequence is not required to be ordered, nor locally stored.
